# 學生使用指南：盲生數學測驗系統

本指南旨在協助視障學生使用 NVDA 螢幕閱讀器順暢地完成數學測驗。

## 1. 登入系統
1. 使用建議瀏覽器（Chrome 或 Edge）開啟系統網址。
2. 輸入 Email 與密碼。
3. 在身份選擇中選擇「學生」後登入。

## 2. NVDA 螢幕閱讀器設定
1. 確認 NVDA 已啟動並正常運作。
2. **瀏覽模式**：使用方向鍵（上/下）朗讀頁面中的文字內容。
3. **焦點模式**：當需要輸入文字或點擊按鈕時，按 Enter 進入該元素。
4. **切換模式**：按 `NVDA + Space` 可在瀏覽模式與焦點模式之間切換。

## 3. 選擇試卷
1. 在首頁使用方向鍵瀏覽目前可用的試卷列表。
2. 找到目標試卷後，按 Enter 進入測驗。

## 4. 作答操作
1. 系統會自動朗讀：題號、題目內容以及數學式的中文朗讀版本。
2. 根據題型進行操作：
   - **選擇題**：使用方向鍵選取選項，按 Enter 確認選擇。
   - **填充題**：按 Enter 進入輸入框，直接輸入答案。
   - **數學計算題**：操作方式與填充題相同。
   - **是非題**：使用方向鍵選取「正確」或「錯誤」。
3. 每當選擇選項時，NVDA 會即時回饋（例如：「你選擇了選項 A」）。

## 5. 導航按鈕
1. 使用 `Tab` 鍵在頁面按鈕之間切換。
2. 常用按鈕包括：「上一題」、「下一題」以及「送出試卷」。
3. NVDA 會朗讀當前進度（例如：「第 3 題，共 10 題」）。

## 6. 查看成績
1. 送出試卷後，系統會自動跳轉至成績頁面。
2. NVDA 會朗讀總分與答對題數。
3. 在錯題區，NVDA 會逐題朗讀錯誤題目與正確答案。

## 7. 鍵盤快捷鍵總覽
- `Tab`：移至下一個元素。
- `Shift + Tab`：移至上一個元素。
- `方向鍵`：在瀏覽模式下閱讀內容。
- `Enter`：啟動按鈕或進入輸入框。
- `NVDA + Space`：切換瀏覽/焦點模式。
- `Escape`：離開輸入框或取消操作。
