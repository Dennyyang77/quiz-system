# 教師使用指南：盲生數學測驗系統

本指南旨在協助教師高效地建立與管理針對視障學生設計的數學試卷。本系統的核心在於提供高品質的 NVDA 朗讀文字，確保盲生能準確理解數學題目。

## 1. 登入系統
1. 開啟瀏覽器進入系統網址。
2. 輸入 Email 與密碼。
3. 在身份選擇中選擇「老師」後登入。

## 2. 建立試卷：手動輸入
1. 進入管理後台。
2. 點擊「新增試卷」。
3. 填寫試卷標題、科目與年級。
4. 使用題目編輯器逐題新增，請根據題型輸入：
   - **選擇題**：輸入題目、A/B/C/D 四個選項，並勾選正確答案。
   - **填充題**：輸入題目與正確答案。
   - **數學計算題**：輸入題目、LaTeX 數學式、NVDA 朗讀文字以及正確答案。
   - **是非題**：輸入敘述，並選擇正確或錯誤。
5. **重要**：每道題目必須填寫「NVDA 朗讀文字」，這是盲生獲取題目內容的唯一途徑。
6. 設定題目難度（簡單/中等/困難）。
7. 儲存試卷。

## 3. 匯入題目：JSON 檔
1. 準備好符合系統規範的 JSON 格式題目檔。
2. 在管理後台點擊「JSON 匯入」。
3. 上傳 .json 檔案。
4. 審核 AI 生成的題目內容。
5. 確認無誤後執行匯入。

## 4. 匯入題目：文字貼上
1. 使用 `math_transfer.py` 工具將 PDF 考卷預先轉換為 NVDA 朗讀文字。
2. 在管理後台點擊「文字貼上匯入」。
3. 將 Word 檔內容複製並貼上至輸入框。
4. 點擊「AI 解析」。
5. 審核解析結果，修改錯誤部分。
6. 確認匯入。

## 5. 匯入題目：PDF 上傳
1. 在管理後台點擊「PDF 匯入」。
2. 上傳 PDF 考卷檔案。
3. 系統 AI 將自動執行轉換。
4. 審核轉換結果並確認匯入。

## 6. 發布試卷
1. 在試卷列表中找到目標試卷。
2. 將狀態更改為「已發布」。
3. 學生登入後即可在首頁看到該試卷並開始作答。

## 7. 查看統計
1. 進入統計頁面查看全班答題狀況。
2. 分析錯題率最高的題目以調整教學。
3. 查看個別學生的表現紀錄。

## 8. NVDA 朗讀文字撰寫要點
為了讓盲生能準確理解數學邏輯，請參考以下轉換規則撰寫朗讀文字：
- 括號：`(` 轉為「前括號」，`)` 轉為「後括號」。
- 運算符：`+` 轉為「加」，`-` 轉為「減」或「負」。
- 分數：`a/b` 轉為「b 分之 a」。
- 次方：`x²` 轉為「x 的 2 次方」。
- 根號：`√` 轉為「根號」。
- 常數：`π` 轉為「拍」。
